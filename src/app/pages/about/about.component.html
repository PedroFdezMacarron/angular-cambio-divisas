conversor de moneda

¿Cómo está desarrollado?

Utiliza angular y lazy loading

¿tiene servicios?

Si. Para las funciones de solicitud de los datos y cálculo. Así como la asignación a las variables amount y lista-divisas

¿Utiliza tipado de variables?

Si. El objeto principal tiene cuatro propiedades para la moneda base la fecha y el importe más una array de objetos con tres propiedades: La divisa, el cambio actual y el importe resultante.

¿Qué url utiliza para solicitar los datos?

la ruta tiene al final una variable de 3 caracteres para indicar la moneda base. la url es: 'https://api.vatcomply.com/rates?base=EUR'.
